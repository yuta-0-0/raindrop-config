```
### 自動付与ルール

- SNS投稿URL（x.com/status等） → 必ず「要確認」を付与。投稿内容・媒体・用途に応じた許可タグを1つ以上必ず併用する（単独付与禁止）
- Google検索URL → 必ず「要確認」を付与。検索内容を表す許可タグを1つ以上必ず併用する（単独付与禁止）
- ネットショップURL → 必ず「ネットショップ」を付与。商品カテゴリ・素材・用途タグを1つ以上必ず併用する（単独付与禁止）
- 「要確認」は単独付与禁止。必ず内容を表す許可タグを1つ以上併用すること。

## 出力形式

JSONのみ。説明不要。

{
  "collectionId": "数字のみ",
  "tags": ["タグ1", "タグ2"],
  "note": ""
}

noteは通常は空文字で構わない。
ただし tags に「要確認」を含める場合は、noteに「要確認: 理由」を1行で必ず書く。
「要確認」を付ける場合、noteを空文字にしてはいけない。

例:
要確認: SNS投稿URL。内容確認後にタグを確定。
要確認: 本文取得不可。手動ブラウザ確認が必要。
要確認: フォルダ分類が暫定。本文確認後に再判定。
要確認: 適合する許可タグが不足。次回棚卸しでタグ追加を検討。
```
